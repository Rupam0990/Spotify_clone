# Spotify Clone 🎵

A front-end clone of Spotify's Web Player built using pure HTML and CSS.

[Click here to view](https://rupam0990.github.io/Spotify_clone/)

## ✨ Features
- Sticky top navigation bar
- Music cards with hover effects
- Fully styled music player at the bottom
- Custom progress bar and volume slider
- Font Awesome icons
- Google Fonts (Montserrat)
- Custom scrollbar styling
- Responsive design with media queries(for sticky nav bar)

## 🛠️ Built With
- HTML5
- CSS3
- Font Awesome 7
- Google Fonts
